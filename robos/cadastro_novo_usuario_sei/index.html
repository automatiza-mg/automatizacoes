    Cadastro de Novo Usuário SEI⚓︎

    Informações gerais⚓︎

    Desenvolvedor Pablo Cezar Moreira Paz - NUSISCORP / SUBDIGITAL / SEPLAG MG
    E-mail atendimentosei@planejamento.mg.gov.br
    Ferramenta Power Automate Desktop
    Versão Power Automate 2.39.00239.23332
    Versão Excel 2016 ou superior
    • Robô dedicado ao cadastro de novo Usuário no SIP / SEI!MG.
    • Indicado para o cadastro múltiplo de usuários.
    • Repete o procedimento a partir do momento do cadastro de novo usuário.
    • Reduz consideravelmente o tempo no procedimento e evita erros.
    Clique para ver o fluxo do robô
    graph TD; 1((Início))-->2; 2[Acesso ao SEI!MG]-->3; 3[Login do Administrador do Órgão/Entidade]-->4; 4[Acesso ao menu Usuários]-->5; 5[Acesso ao menu Novo]-->6; 6[Preenchimento de informações: Órgão/Entidade Logado, Sigla, Nome]-->7; 7[Loop a partir do menu Novo* até o último usuário informado na planilha.]-->8; 8((Fim))

    Pré-requisitos⚓︎

    Power Automate na versão correta

    Chrome como navegador

    Extensão Chrome para Power Automate ativada.

    Montando o seu robô⚓︎

    Este é um protótipo de robô. Entenda seu funcionamento e, caso precise, ajuste-o às suas necessidades.

    Para dar vida a ele:

    Copie o código do robô1 e cole em um novo fluxo Power Automate Desktop.

    Clique aqui para baixar a planilha modelo2.

    Crie a variável de entrada local_excel com o caminho onde a planilha modelo foi salva.

    Crie a variável de entrada login_sei.

    Crie a variável de entrada senha_sei.

    Crie a variável de entrada orgao_sei.

    Caso precise de ajuda

    Algum erro está ocorrendo em uma das fases de criação de seu robô? Não tem problema. Acesse aqui uma lista com causas de erros ou paradas do robô.

    Também temos o espaço abaixo para perguntas e respostas. Dê uma olhadinha lá! Quem sabe alguém já não fez uma pergunta com o mesmo problema que você está enfrentando?

    Caso prefira, mande um e-mail para simplificacao@planejamento.mg.gov.br.

    Ótimo trabalho!

    1. Na nova aba que será aberta, basta apertar Ctrl+A para selecionar todo código e Ctrl+C para copiar. ↩

    2. Qualquer modificação além da inclusão de mais usuários na planilha Excel pode exigir modificações no código original do robô. Download desta planilha não é suportado na versão mobile desta página. ↩
