# Já se deparou com o erro 'ReadAsText' no Power Automate desktop? Veja aqui como resolver!

Esse erro pode aparecer quando você cola um código no seu fluxo do Power Automate. Mais especificamente, com o texto `Argumentos desconhecidos: 'ReadAsText'.`. 
Continue lendo para saber como resolvê-lo!


<!-- more -->
## Erros ao colar um código no Power Automate desktop

Geralmente, os erros que acontecem no Automate desktop quando se cola um código no fluxo se referem a parâmetros faltantes - como, por exemplo, variáveis de entrada ainda não criadas, subfluxos faltantes ou caminhos de arquivo ainda não preenchidos. Todos esses se resolvem quando fazemos exatamente o que a mensagem de erro sugere, ou seja, completar os parâmetros faltantes. 

No entanto, o erro de 'ReadAsText' não é tão indicativo. Ele não dá orientações ou indícios do que fazer. 

## Como resolver então?

Se algum dia esse erro aparecer quando você colar um código, **vá até a ação indicada na mensagem de erro**, **abra** essa ação e depois a **feche**. Sem edição, sem modificação nos parâmetros. Apenas abrir e fechar a ação indicada.

Isso tende a fazer o erro desaparecer da sua tela de avisos do Automate.
