# Automação de processos de autuação de infração de trânsito

Este projeto envolveu o processo de recebimento, análise e encerramento de recursos autuação de infração de trânsito (AIT) da Coordenação Estadual de Trânsito de Minas Gerais (CET). 

O processo ocorre diariamente e envolve os sistemas / plataformas **Microsoft Excel**, **SIAUT**, **SDAK** e **SEI**. 

Foram desenvolvidas **as seguintes automatizações:**

1. Robô de Recepção de Protocolo Digital
2. Robô de Recepção física 
3. Robô de procura de AIT
4. Robô de Encerramento
5. Robô de análise de recurso
