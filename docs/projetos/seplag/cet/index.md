# Automação de processos de auto de infração de trânsito

Este projeto envolveu o processo de recebimento, análise e encerramento de recursos auto de infração de trânsito (AIT) da Coordenação Estadual de Trânsito de Minas Gerais (CET). 

O processo ocorre diariamente e envolve os sistemas / plataformas **Microsoft Excel**, **SIAUT**, **SDAK** e **SEI**. 

Foram desenvolvidas **as seguintes automatizações:**

1. Robô de recepção de protocolo digital
2. [Robô de montagem de defesa recebida no protocolo digital](montagem_defesa_digital.md)
2. Robô de recepção física 
3. Robô de procura de AIT
4. Robô de encerramento
5. Robô de análise de recurso
6. Robô de desbloqueio de gravame
