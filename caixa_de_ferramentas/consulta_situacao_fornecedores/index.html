    Consulta Situação Cadastral de Fornecedores - CAFIMP⚓︎

    • Responsável: Isabela Romancini.
    • Data do treinamento: 28/09/2023
    • Local: Sala SEE, Torre Minas, 11º andar
    • Turma treinada: órgãos diversos
    • Arquivo auxiliar: consulta-situacao-cadastral-fornecedores.xlsx

    O robô para consulta de situação cadastral de fornecedores foi desenvolvido com a ferramenta Microsoft Power Automate e pode auxiliar na automação de tarefas relacionadas à gestão de compras e fornecedores. Em linhas gerais ele é capaz de:

    Funções desenvolvidas pelo protótipo:⚓︎

    • Ler uma planilha, em formato Excel, contendo CNPJs de fornecedores.
    • Consultar a situação cadastral desses fornecedores em diversos sites governamentais.
    • Atualizar a situação cadastral desses fornecedores na mesma planilha Excel aberta no início do processo, determinando se eles estão impedidos ou não de participar de licitações com a administração pública.
    • Salvar documentos (imagens e pdfs) que comprovem a consulta.

    Essa automação oferece um processo eficiente de atualização da lista de fornecedores, economizando tempo em processos de compras e gestão de contratos.

    Montando o seu robô⚓︎

    Em primeiro lugar, é importante relembrar que o conteúdo desta página é um protótipo de robô. Portanto, se você pretende utilizar este protótipo, tenha especial atenção às especificidades do problema que você quer solucionar, ajustando o código às suas necessidades.

    O arquivo contendo o código deste robô está armazenado nesta página. Para dar vida ao seu, clique na caixa copiar e colar abaixo, copie e cole o código disponível em um novo fluxo Power Automate Desktop. Não se esqueça de personalizar seu novo robô, conforme orientado abaixo.

    Consulta Situação Cadastral de Fornecedores CAFIMP 
    SET linha TO 2
Excel.LaunchExcel.LaunchAndOpenUnderExistingProcess Path: Caminho_excel Visible: True ReadOnly: False Instance=> ExcelInstance
Excel.ReadFromExcel.ReadCell Instance: ExcelInstance StartColumn: $'''A''' StartRow: linha ReadAsText: False CellValue=> CNPJ
LOOP WHILE (CNPJ) > (0)
    WebAutomation.LaunchChrome.LaunchChrome Url: $'''https://www.cagef.mg.gov.br/fornecedor-web/br/gov/prodemge/seplag/fornecedor/publico/index.zul''' WindowState: WebAutomation.BrowserWindowState.Maximized ClearCache: False ClearCookies: False WaitForPageToLoadTimeout: 60 Timeout: 60 BrowserInstance=> Browser
    MouseAndKeyboard.MoveMouseToImage.ClickImage Images: [imgrepo['Menu']] SearchForImageOn: MouseAndKeyboard.SearchTarget.EntireScreen MousePositionOnImage: MouseAndKeyboard.PositionOnImage.MiddleCenter OffsetX: 0 OffsetY: 0 Tolerance: 10 MovementStyle: MouseAndKeyboard.MovementStyle.Instant Occurence: 1 Timeout: 5 ClickType: MouseAndKeyboard.ClickType.LeftClick SecondsBeforeClick: 0 SearchAlgorithm: MouseAndKeyboard.ImageFinderAlgorithm.Legacy X=> X Y=> Y
    MouseAndKeyboard.SendKeys.FocusAndSendKeys TextToSend: $'''{Tab}{Tab}{Tab}{Return}{Tab}{Tab}{Tab}{Return}''' DelayBetweenKeystrokes: 10 SendTextAsHardwareKeys: False
    WAIT (WebAutomation.WaitForWebPageContent.WebPageToContainText BrowserInstance: Browser Text: $'''CNPJ do Fornecedor''')
    MouseAndKeyboard.SendKeys.FocusAndSendKeys TextToSend: $'''{Tab}{Tab}%CNPJ%''' DelayBetweenKeystrokes: 50 SendTextAsHardwareKeys: False
    MouseAndKeyboard.MoveMouseToImage.ClickImage Images: [imgrepo['Pesquisar']] SearchForImageOn: MouseAndKeyboard.SearchTarget.EntireScreen MousePositionOnImage: MouseAndKeyboard.PositionOnImage.MiddleCenter OffsetX: 0 OffsetY: 0 Tolerance: 10 MovementStyle: MouseAndKeyboard.MovementStyle.Instant Occurence: 1 Timeout: 5 ClickType: MouseAndKeyboard.ClickType.LeftClick SecondsBeforeClick: 0 SearchAlgorithm: MouseAndKeyboard.ImageFinderAlgorithm.Legacy X=> X2 Y=> Y2
    WebAutomation.ExtractData.ExtractSingleValue BrowserInstance: Browser ExtractionParameters: {[$'''html > body > div:eq(0) > div > div > div > div > div > div:eq(1) > div:eq(1) > div > div > div > div > div:eq(1) > div:eq(0) > div:eq(1) > div > div:eq(3) > div > span''', $'''Own Text''', $''''''] } TimeoutInSeconds: 60 ExtractedData=> textocafimp
    Clipboard.SetText Text: textocafimp
    Clipboard.GetText Text=> textoparaexcel
    IF Contains(textoparaexcel, $'''NÃO SE ENCONTRA INSCRITO''', True) THEN
        Excel.WriteToExcel.WriteCell Instance: ExcelInstance Value: $'''Não se encontra inscrito''' Column: $'''B''' Row: linha
    ELSE
        Excel.WriteToExcel.WriteCell Instance: ExcelInstance Value: $'''Inscrito''' Column: $'''B''' Row: linha
    END
    # Crie uma pasta no seu computador chamada "Automatiza" e inclua seu caminho na variável "Caminho_pasta_automatiza"
    Folder.Create FolderPath: Caminho_pasta_automatiza FolderName: CNPJ Folder=> pastacnpj
    WebAutomation.TakeScreenshot.TakeScreenshotSaveToFile BrowserInstance: Browser File: $'''%Caminho_pasta_automatiza%%CNPJ%\\CAFIMP.png''' FileFormat: WebAutomation.FileFormat.Png
    WebAutomation.CloseWebBrowser BrowserInstance: Browser
    Variables.IncreaseVariable Value: linha IncrementValue: 1
    Excel.ReadFromExcel.ReadCell Instance: ExcelInstance StartColumn: $'''A''' StartRow: linha ReadAsText: False CellValue=> CNPJ
END
Excel.CloseExcel.CloseAndSave Instance: ExcelInstance

    • Revise todas instâncias criadas (nomes dos arquivos, páginas da web, etc).

    • Revise e edite todas as variáveis criadas (ex.: nomes das colunas que serão criadas/preenchidas, etc).

    • Confira o caminho dos arquivos editados pelo robô no seu computador (PATH)

    • Certifique-se de haja ações para todas as operações desejadas que o robô execute
